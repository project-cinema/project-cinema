# project cinema

## setup

### windows
* install wsl
  * https://qiita.com/toshi772/items/05f3c7e3a99df42d4cec
* install ubuntu on wsl
  * wsl --install -d Ubuntu
* connect wsl by intellij idea
* setup on ubuntu

### ubuntu

* get token 
  * https://github.com/settings/tokens
* run script
```bash
sh setup.sh
```
